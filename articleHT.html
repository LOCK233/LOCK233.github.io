HTML笔记

标签

一、基础标签

  • <h1>-<h6>标题标签 (head) eg:<h1>我是一级标签</h1>
  • <p></p>段落标签 （paragraph）
  • <br />换行标签 （break）
  • <strong> </strong>（更强烈） 或<b></b> 加粗标签
  • <em></em>（更强烈） 或<i></i> 倾斜标签
  • <del></del>（更强烈） 或<s></s>> 删除线
  • <ins></ins>（更强烈） 或<u></u> 下划线
  • <div><span>无语义装内容

二、图像标签

图像标签和路径（相对绝对） <img src="图像URL"/> <img src="" alt="" title="" width="" height="" border="">

  • src 图片路径 必须属性
  • alt 文本 替换不能显示的文本为文字
  • title 文本 提示文本，鼠标放在图像上，显示的文字
  • width 像素 设置图像宽度
  • height 像素 设置图像高度
  • border 像素 设置图像边框粗细

三、链接标签

1.<a herf ="A">xxxx</a> 则点击xxxx跳转至A（内部html文件跳转）， 2.如A等于外部网址则为外部链接 如写target="_blank"于"A"后面则表示新开一个网页跳转 3.<a herf ="#">xxxx</a> 跳转空页面 4.<a herf ="">下载文件</a> 其中地址的链接为exe或zip等压缩包形式 5.<a herf="网址">``<img src="图片链接"/>``</a>图片跳转网址

四、锚点链接

锚点链接 点完快速跳转到页面中某一位置

  • 在链接文本的herf属性中，设置属性值为#名字的形式eg:<a herf="#名字">xxx</a>
  • 找到目标标签位置标签，里面添加一个id属性=刚刚的名字，如：<h3 id="two">xxx介绍</h3>

五、注释标签

注释标签<!-- xxx -->打出ctrl+/可取消(或怎加)注释标签 （特殊字符见Q）

六、表格的基本语法

<table>(table是用来定义表格的标签)
<tr>（tr定义行，嵌套于table中）
<td>单元格中文字`</td>`（td定义单元格，必须嵌套于tr中）
 ...
</tr>
    ...
</table>

<th>代表表头单元格，写出来文本会居中并加粗（表格属性名见Q，这些写到标签table里面去） <thead>表示表格的头部区域 <tbody>表示表格的主题区域

合并单元格 跨行合并单元格：rowspan="合并单元格的个数“ 跨列合并单元格：colspan="合并单元格的个数"

目标单元格 跨行：最上侧为目标单元格 跨列：最左侧为目标单元格 写在td中 合并完删除多余单元格

七、无序列表

<ul>表示无序列表，一般以项目符号呈现列表项，而列表项用<li>标签定义 基本语法 :

<ul> <li>列表项1</li> <li>列表项2</li> <li>列表项3</li> ...... </ul>

  • ul标签中只能放li标签

  • li中可以放任何元素

八、有序列表

<ol>表示无序列表，一般以项目符号呈现数字，而列表项用<li>标签定义 基本语法

<ol> <li>列表项1</li> <li>列表项2</li> <li>列表项3</li> ...... </ul> life-style：none 可去除小圆点

九、自定义列表

定义列表的列表项前没有任何项目符号，用于定义描述列表，与<dt>（定义项目/名字）和<dd>（描述每一个项目/名字）一起使用 基本语法

<dl> <dt>名词1</dt> <dd>名词2解释1</dd> <dd>名词3解释2</dd> </dl>

注：<dl>``</dl>只能包含<dt>和<dd> <dt><dd>无个数限制，经常是一个<dt>对应多个<dd>

十、表单

1.表单域

表单由 表单域 表单控件 提示信息组成 <form>定义表单域，收集用户信息并传递给服务器 格式：<form action="url地址" method="提交方式" name=”表单域名称“>

属性 属性值 作用
action url地址 指定地址
method get/post 提交方式设置
name 名称 指定名称区分不同表单域

2.input表单元素

<input表单元素 格式:<input type="属性值"/> type用于设置不同属性值以指定不同控件类型 （可取值见Q） 除type属性外，<input>标签还有其他属性，常用属性如下

属性 属性值 描述
name 自定义 定义input元素名称
value 自定义 规定input元素的值
checked checked 规定input首次加载时应当被选中
maxiength 正整数 规定输入字段中字符的最大长度

3.label标签

<label>标签绑定一个表签文本，浏览器会自动将焦点（光标）转到或者选择相应的表单元素，用来增加用户体验

eg:<lable for="sex">男</label> <input type="radio" name="sex id="sex"/>

4.select 下拉表单标签

<select>
 <option>选项1</option>
 <option>选项2</option>
 <option>选项3</option>
  ......
</select>
  • 在option中写入selected="selected"时当前项即为默认选中项

textarea表单元素

在用户输入内容较多的情况下就不能使用文本框表单，此时我们可以使用<textarea>标签 格式：

<textarea row="3" cols="20">(rows=显示行数 cols=每行字符数) 文本内容 </textarea>
