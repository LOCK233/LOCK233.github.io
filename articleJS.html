作用域

1.JavaScript作用域就是代码名字(变量)在某个范围内起的效果和作用，提升程序的可靠性，减少命名冲突 2.js的作用域：全部作用域 局部作用域 3.全局作用域：整个script标签，或者是一个单独的js文件 （在函数内部没有定义的变量也看作全局变量） 4.局部作用域(函数作用域)在函数内部就是局部作用域 这个代码的名字旨在函数内部起效果和作用。（函数的形参为局部变量） 5.作用域链：内部函数访问外部函数的变量，采取的是链式查找的方式来决定取那个值，这种结构我们称作作用域链，采用就近原则

对象

在JavaScript中，对象是一组无序的相关属性和方法的集合，所有的事物都是对象，例如字符串、数值、数组、函数等 对象是由属性和方法组成的

属性：事物的特征，在对象中用 属性 来表示(常用名词) 方法：事物的特征，在对象中用 方法来表示(常用动词)

1.利用对象自变量创建对象{}

语法格式 var obj = { //对象的属性与方法 }

里面的属性或者方法我们采取键值对的形式 键 属性名；值 属性值 多个属性或者方法中间用逗号隔开 方法后面跟的是一个匿名函数

2.调用对象属性

采取对象名+属性的方法 eg:console.log(obj.uname)

法二：对象名['属性名']

3.new Object 创建对象

语法结构 var 对象名 = new Object(){ 对象名.属性名 = '值'； 对象名.方法名 = function(){ //行为 } }

(1)利用等号赋值的方法添加对象的属性和方法 (2)每个属性和方法之间用分号结束

3.利用构造函数创建对象

(1)构造函数名字首字母要大写 (2)构造函数不需要return就可以返回结果 (3)我们调用构造函数 必须使用new (4)我们只要new 函数名() 调用函数就创建一个对象 (5)我们的属性和方法前面必须添加this

4.遍历对象属性

JS中的简单数据类型与复杂数据类型

1.简单数据类型，复杂数据类型

2.堆和栈

(1)堆(操作系统)：由操作系统自动分配释放函数的参数值、局部变量的值等。其操作方式类似于数据结构中的栈；简单数据类型存放到栈里面 (2)堆(操作系统)：存储复杂类型(对象)，一般由程序员分配释放，如程序员不释放，由垃圾回收机制回收。复杂数据类型存放到堆里面

简单数据类型存放在栈里面，直接开辟一个空间，存放的是值 复杂数据类型首先在栈里面存放地址，用十六进制表示，然后地址指向堆里面的数据
